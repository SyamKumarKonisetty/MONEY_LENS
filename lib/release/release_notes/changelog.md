# Changelog

All notable changes to the MoneyLens project will be documented in this file. MoneyLens V2 adheres to Semantic Versioning (`vMAJOR.MINOR.PATCH`).

## [1.2.0] — 2026-06-20

### Added
- **Security & Reliability Framework (SRF)**: Progressive keypad lockouts, secure data parsing isolate threads, atomic Drift transactions, SMS sandbox filters, and database corruption recovery triggers.
- **Financial Intelligence Framework (FIF)**: Decoupled health indicators, budget runway calculations, comparison engines, and 20 interactive chart visualizations.
- **Premium Polish Phase (PPP)**: Unified `MLHaptics` trigger integrations, refined visual blurs, OLED Pure Black backdrop settings for dark mode, and improved transition timings.
- **Release Candidate Framework (RCF)**: Production build files verification, Android target SDK upgrade, and full compliance metadata templates.

### Changed
- Refactored `auth_provider.dart` to support secure SHA-256 PIN hashes and automated migrations.
- Offloaded heavy CSV parsing and serializations to background isolates.
- Standardized UI typography tokens using the Financial Typography System (FTS).

## [1.1.0] — 2026-06-18

### Added
- **Smart Inbox 2.0**: Automated transaction detection from incoming SMS bank notifications.
- **Budget Rings**: Visual progress indicators for active budget limits.
- **Theme Engine**: Toggle between Light Mode and Dark Mode in settings.

## [1.0.0] — 2026-06-14

### Added
- **MoneyLens v1.0 Launch**: Initial stable release featuring local transaction registry, custom category configurations, and basic CSV exports.

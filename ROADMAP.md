# MoneyLens V2 — Feature Roadmap

This roadmap outlines the long-term plan for the evolution of the MoneyLens application.

## Phase 1: Security Hardening & Stable Release (v1.2.0)
- Complete database validation, backup sanitizations, and local PIN security.
- Finalize the RCF (Release Candidate Framework) for initial Google Play submission.

## Phase 2: User Experience and Visual Delight (v1.3.0)
- Expand the MLDS component catalog (MLButton, MLCard, MLInput implementations replacing stubs).
- Introduce advanced micro-animations for dashboard greetings and transaction swipe deletes.

## Phase 3: Analytics Extensions (v1.4.0)
- Integrate customizable chart ranges and multi-month comparison reports.
- Support recurring budget intervals (weekly, quarterly, annually).

## Phase 4: Local Smart Integrations (v2.0.0)
- Introduce localized AI-driven transaction categorization running entirely on-device (via local TF Lite models).
- Add support for cross-device manual encrypted sync over local Wi-Fi networks (peer-to-peer, no cloud).

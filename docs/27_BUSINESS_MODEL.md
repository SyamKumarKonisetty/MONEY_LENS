# 27: Business Model

## Purpose
To explain how MoneyLens maintains financial viability without harvesting and selling user data.

## The Problem with the Industry
Competitors are "free" because the user is the product. Their servers parse transaction data to target ads or sell anonymized spending behaviors.

## MoneyLens Revenue Model
MoneyLens fundamentally rejects ad-tech. 

### Current Status (Free Tier)
V1 is entirely free. It establishes trust, builds a user base, and proves the utility of the UI Engine.

### Future Status (Premium Tier)
MoneyLens will introduce a single, lifetime purchase (or a very low-cost annual subscription) that unlocks:
- Cross-device encrypted sync (User provides their own Google Drive).
- PDF Customization (Branded export reports).
- Custom Themes / Color Palettes.
- Unlimited savings goals.

The core SMS tracking and privacy features will remain free forever.
